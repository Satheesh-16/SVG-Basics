    • M (Move To): Moves the pen to a new position without drawing (absolute/relative).
    • L (Line To): Draws a straight line from the current point to a new point (absolute/relative).
    • C (Curve To): Draws a cubic BÃ©zier curve (absolute/relative).
    • A (Arc): Draws an elliptical arc curve (absolute/relative).
    • Z (Close Path): Closes the path by drawing a line from the current point to the starting point.
