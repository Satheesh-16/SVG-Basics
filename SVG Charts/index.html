Talk About SVG

  • Scalable Vector Graphics
  • XML based Markup language on 2d vector graphics
  • No loss of Quality
  • Developed by World Wide Web Consortium in 1999
  • Supported by all major browsers
  • Like a geometry concerned specific Graph
  • More image to svg converters available in internet.

Why SVG

  • SEO Friendly
  • Scalability
  • Small File Size
  • Variety Of Shapes and Path
  • Mathematical Precision
  • Editable and user friendly
